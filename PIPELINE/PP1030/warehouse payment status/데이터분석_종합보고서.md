## 3. 창고비용 배분 방식 제안

### 3.1 배분 기준 옵션

#### Option A: SQM × 일수 (권장)
**공식:**
```
개별 아이템 창고비용 = (해당월 창고 총비용) × (아이템 SQM × 체류일수) / (해당월 전체 SQM×일수 합계)
```

**장점:**
- 공간과 시간 모두 고려한 정확한 배분
- 실제 창고 사용량 반영

**단점:**
- 계산 복잡
- 체류기간 없는 아이템 처리 필요

#### Option B: SQM만 사용 (간단)
**공식:**
```
개별 아이템 창고비용 = (해당월 창고 총비용) × (아이템 SQM) / (해당월 전체 SQM 합계)
```

**장점:**
- 계산 단순
- 직관적 이해 가능

**단점:**
- 장기/단기 체류 구분 없음

#### Option C: HVDC 코드 그룹별 배분
**방식:**
- 같은 HVDC 코드 내에서 아이템 SQM 비율로 배분
- 월별로 HVDC 코드별 창고비용 집계 후 배분

### 3.2 비용 배분 대상 선택

| 컬럼 | 설명 | 권장 여부 |
|------|------|----------|
| `Amount` | 창고 임대료만 | ✅ 권장 (Storage 비용) |
| `TOTAL` | 전체 창고 비용 | ⚠️ Handling 비용 포함됨 |

### 3.3 미매칭 데이터 처리

**창고 코드만 있는 경우 (예: DSV Open Yard, M44 Warehouse):**
- 해당 월의 평균 단가 적용
- 또는 별도 "창고 공통비용"으로 분리

**체류기간 없는 아이템 (5,558건):**
- Option 1: 창고비용 0 처리
- Option 2: 평균 체류기간으로 추정 배분
