# ③ YAML 자동 생성 + 백업 VBA

### 📌 생성 규칙

* 파일명: `pipeline_config.yaml`
* 백업: `pipeline_config_YYYYMMDD_HHMMSS.yaml`

---

### VBA 모듈: `modConfigYaml`

```vba
Option Explicit

Sub BTN_SaveConfig()
    Dim cfgPath As String, bakPath As String
    Dim ts As String, f As Integer

    cfgPath = NM_CONFIG_DIR.Value & "\pipeline_config.yaml"
    ts = Format(Now, "yyyymmdd_hhnnss")
    bakPath = NM_CONFIG_DIR.Value & "\pipeline_config_" & ts & ".yaml"

    ' 기존 config 백업
    If Dir(cfgPath) <> "" Then
        FileCopy cfgPath, bakPath
    End If

    f = FreeFile
    Open cfgPath For Output As #f

    Print #f, "project:"
    Print #f, "  name: " & NM_PROJECT_NAME.Value
    Print #f, "  env: " & NM_ENV.Value
    Print #f, ""
    Print #f, "paths:"
    Print #f, "  root: " & NM_PIPELINE_ROOT.Value
    Print #f, ""
    Print #f, "stages:"
    Print #f, "  stage1:"
    Print #f, "    master: " & NM_S1_MASTER.Value
    Print #f, "    warehouse: " & NM_S1_WH.Value
    Print #f, "    output: " & NM_S1_OUT.Value
    Print #f, "  stage2:"
    Print #f, "    input: " & NM_S2_IN.Value
    Print #f, "    output: " & NM_S2_OUT.Value
    Print #f, "  stage3:"
    Print #f, "    input: " & NM_S3_IN.Value
    Print #f, "    output: " & NM_S3_OUT.Value
    Print #f, "  stage4:"
    Print #f, "    input: " & NM_S4_IN.Value

    Close #f
    MsgBox "YAML config saved + backup created", vbInformation
End Sub
```

# ⑤ **Stage3 최신 리포트 → Stage4 publish 버튼**

### 로직

1. Stage3 Output 폴더에서
   `HVDC_입고로직_종합리포트_*.xlsx` 검색
2. **최종 수정일 가장 최신 파일 선택**
3. CONTROL!B20(Stage4 Input)에 경로 자동 입력
4. YAML 재저장

---

### VBA: `BTN_PublishStage3To4`

```vba
Sub BTN_PublishStage3To4()
    Dim fso As Object, fld As Object, fil As Object
    Dim latestFile As String, latestTime As Date

    Set fso = CreateObject("Scripting.FileSystemObject")
    Set fld = fso.GetFolder(NM_S3_OUT.Value)

    latestTime = #1/1/2000#
    latestFile = ""

    For Each fil In fld.Files
        If fil.Name Like "HVDC_입고로직_종합리포트_*.xlsx" Then
            If fil.DateLastModified > latestTime Then
                latestTime = fil.DateLastModified
                latestFile = fil.Path
            End If
        End If
    Next fil

    If latestFile = "" Then
        MsgBox "No Stage3 report found.", vbCritical
        Exit Sub
    End If

    NM_S4_IN.Value = latestFile
    Call BTN_SaveConfig

    MsgBox "Published latest Stage3 report to Stage4 input:" & vbCrLf & latestFile, vbInformation
End Sub
```
