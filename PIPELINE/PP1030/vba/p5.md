According to a document from **2025-12-21**, Stage 3/4 파이프라인은 `pipeline_config.yaml`의 **stage3.io(derived_file/report_file)**, 그리고 Stage 4는 `stage4_anomaly.yaml`의 **stage4.io(input_file/sheet_name/output)**를 기준으로 동작하도록 정의돼 있습니다.

아래는 요청하신 대로 **“Excel CONTROL 시트만으로 운영(=YAML 직접 편집 없이)”** 하도록, **CONTROL 시트 템플릿(셀 주소/이름정의/버튼명 고정)** + **VBA 버튼 운영** + **YAML 자동 반영(백업 포함)** + **Stage1~4 실행 버튼** + **Stage3 최신 리포트 자동 탐색→Stage4 input publish 버튼** + **에러코드→MsgBox 매핑** + **LED(🟢🟡🔴) 상태보드**까지 한 번에 구현 가능한 고정 설계안입니다.

---

## 1) 핵심요약(2–4줄)

* **CONTROL 시트**에 경로/옵션을 입력하면, **VBA 버튼**이 **Python `run_pipeline.py`**를 호출해 **Stage1~4**를 실행합니다.
* CONTROL 값은 버튼 1개로 **YAML(config) 자동 갱신 + 백업**되며, 사용자는 **YAML 파일을 직접 열 필요가 없습니다**.
* Stage3 결과물(리포트) 중 **가장 최신 파일을 자동 탐색**해 Stage4 입력 파일로 **publish(복사/덮어쓰기+백업)** 하는 버튼을 제공합니다.
* 각 단계는 **Exit Code → Excel MsgBox**로 즉시 안내하고, CONTROL 상태보드에 **🟢🟡🔴 LED**와 **마지막 실행시각/결과**를 기록합니다.

---

## 2) 구조(시각화)

### 전체 흐름(버튼-운영형)

```
[CONTROL 시트: 경로/옵션 입력]
        ↓ (VBA 버튼)
[VBA: 저장/검증/LED 갱신/로그기록]
        ↓ (WScript.Shell.Exec)
[Python: scripts\run_pipeline.py]
        ↓
 Stage1 (동기화) → Stage2 (파생컬럼) → Stage3 (리포트) → (Publish 최신리포트) → Stage4 (이상치)
        ↓
[결과파일 + logs + Excel LOG 시트]
```

* Stage3 출력 리포트는 `data/processed/reports/HVDC_입고로직_종합리포트_*.xlsx` 패턴(12개 시트)로 정의되어 있고, 그중 `통합_원본데이터_Fixed` 시트(표준 헤더 63개)가 포함됩니다.
* Stage4는 Stage3 리포트의 **`통합_원본데이터_Fixed`**를 읽어 이상치를 생성하고, `stage4_anomaly.yaml`의 `stage4.io`에 input/output이 정의됩니다.
* Stage2는 `stage2_derived_config.yaml`에 `input.synced_file`, `output.derived_file`가 정의됩니다.

'========================
' CORE RUNNER
'========================
Private Function RunAction(ByVal action As String, ByVal boardStage As String, ByVal runStage As String, Optional ByVal silent As Boolean = False) As Long
    On Error GoTo EH

    Dim wsCtl As Worksheet: Set wsCtl = ThisWorkbook.Worksheets(SHEET_CONTROL)

    ' Ensure workbook saved so Python can read latest CONTROL values
    ThisWorkbook.Save

    SetLED boardStage, LED_RUNNING
    SetBoard boardStage, Now, "Running...", ""

    Dim pythonExe As String, runner As String, root As String
    pythonExe = Nz(Range("n_PythonExe").Value)
    runner = Nz(Range("n_RunPipelinePy").Value)
    root = Nz(Range("n_ProjectRoot").Value)

    If Len(root) = 0 Then root = ThisWorkbook.Path
    Dim absRunner As String: absRunner = ToAbsPath(root, runner)

    Dim cmd As String
    cmd = BuildCmd(pythonExe, absRunner, action, runStage, ThisWorkbook.FullName)

    Dim outText As String, errText As String, exitCode As Long
    exitCode = ExecCapture(cmd, outText, errText)

    Dim d As Object: Set d = ParseKeyValue(outText)
    Dim msg As String, artifact As String
    msg = IIf(d.Exists("MESSAGE"), d("MESSAGE"), Left$(outText, 300))
    artifact = IIf(d.Exists("ARTIFACT"), d("ARTIFACT"), "")

    RunAction = exitCode

    If exitCode = 0 Then
        SetLED boardStage, LED_OK
        SetBoard boardStage, Now, msg, artifact
        LogRow action, boardStage, exitCode, msg, artifact, cmd
        If Not silent Then MsgBox boardStage & " 완료" & IIf(Len(artifact) > 0, vbCrLf & artifact, ""), vbInformation
    Else
        SetLED boardStage, LED_FAIL
        SetBoard boardStage, Now, msg & IIf(Len(errText) > 0, vbCrLf & "[stderr] " & Left$(errText, 500), ""), artifact
        LogRow action, boardStage, exitCode, msg, artifact, cmd
        If Not silent Then MsgBox MapExitCode(exitCode, boardStage) & vbCrLf & msg, vbCritical
    End If

    Exit Function

EH:
    RunAction = 90
    SetLED boardStage, LED_FAIL
    SetBoard boardStage, Now, "VBA Exception: " & Err.Description, ""
    LogRow action, boardStage, 90, "VBA Exception: " & Err.Description, "", ""
    If Not silent Then MsgBox "VBA 오류: " & Err.Description, vbCritical
End Function

Private Sub SetBoard(ByVal stage As String, ByVal dt As Date, ByVal message As String, ByVal artifact As String)
    Dim r As Long: r = BoardRow(stage)
    With ThisWorkbook.Worksheets(SHEET_CONTROL)
        .Cells(r, "G").Value = Format(dt, "yyyy-mm-dd hh:nn:ss")
        .Cells(r, "H").Value = message

        ' Put artifact as hyperlink if file exists
        If Len(artifact) > 0 Then
            On Error Resume Next
            .Cells(r, "H").Hyperlinks.Delete
            On Error GoTo 0
            If FileExists(artifact) Then
                .Hyperlinks.Add Anchor:=.Cells(r, "H"), Address:=artifact, TextToDisplay:=message & vbCrLf & artifact
            End If
        End If
    End With
End Sub

Private Function BoardRow(ByVal stage As String) As Long
    Select Case UCase$(stage)
        Case "CONFIG_SYNC": BoardRow = 5

Public Function Nz(ByVal v As Variant) As String
    If IsError(v) Then Nz = "": Exit Function
    If IsNull(v) Then Nz = "": Exit Function
    If Len(Trim$(CStr(v))) = 0 Then Nz = "" Else Nz = Trim$(CStr(v))

Public Function Quote(ByVal s As String) As String
    Quote = """" & s & """"

Public Function IsAbsPath(ByVal p As String) As Boolean
    p = Nz(p)
    If Len(p) = 0 Then IsAbsPath = False: Exit Function
    If InStr(1, p, ":\", vbTextCompare) = 2 Then IsAbsPath = True: Exit Function ' C:\...
    If Left$(p, 2) = "\\" Then IsAbsPath = True: Exit Function ' UNC
    IsAbsPath = False
End Function

Public Function ToAbsPath(ByVal root As String, ByVal p As String) As String

        ToAbsPath = p
    Else

        Else

    End If
End Function

Public Function ExecCapture(ByVal cmd As String, ByRef outText As String, ByRef errText As String) As Long

    Set ex = sh.Exec(cmd)

' Parse KEY=VALUE lines to Dictionary
Public Function ParseKeyValue(ByVal text As String) As Object
    Dim d As Object: Set d = CreateObject("Scripting.Dictionary")
    Dim lines() As String, i As Long, p As Long, k As String, v As String
    lines = Split(text, vbCrLf)
    For i = LBound(lines) To UBound(lines)
        p = InStr(1, lines(i), "=", vbBinaryCompare)
        If p > 1 Then
            k = Trim$(Left$(lines(i), p - 1))
            v = Trim$(Mid$(lines(i), p + 1))
            If Len(k) > 0 Then d(k) = v
        End If
    Next i

def _yaml_backend():
    """
    Prefer ruamel.yaml (preserve formatting), fallback to PyYAML.
    """
    try:
        from ruamel.yaml import YAML  # type: ignore
        y = YAML()
        y.preserve_quotes = True
        return ("ruamel", y, None)
    except Exception:
        try:
            import yaml  # type: ignore
            return ("pyyaml", None, yaml)
        except Exception as e:
            raise RuntimeError(
                "YAML 라이브러리가 없습니다. 아래 중 1개 설치하세요:\n"
                "  pip install ruamel.yaml\n"
                "  pip install pyyaml"
            ) from e


def yaml_load(path: Path) -> Any:
    backend, ruamel_obj, pyyaml = _yaml_backend()
    if not path.exists():
        return {}
    if backend == "ruamel":
        with path.open("r", encoding="utf-8") as f:
            return ruamel_obj.load(f) or {}
    else:
        with path.open("r", encoding="utf-8") as f:
            return pyyaml.safe_load(f) or {}


def yaml_dump(data: Any, path: Path) -> None:
    backend, ruamel_obj, pyyaml = _yaml_backend()
    path.parent.mkdir(parents=True, exist_ok=True)
    if backend == "ruamel":
        with path.open("w", encoding="utf-8") as f:
            ruamel_obj.dump(data, f)
    else:
        with path.open("w", encoding="utf-8") as f:
            pyyaml.safe_dump(data, f, sort_keys=False, allow_unicode=True)

    cur = d

        cur = cur[k]
    cur[keys[-1]] = value


def deep_replace_values(obj: Any, replace_map: Dict[str, str]) -> Any:
    """
    Recursively replace exact string values found in obj.
    """
    if isinstance(obj, dict):
        for k, v in list(obj.items()):
            obj[k] = deep_replace_values(v, replace_map)
        return obj
    if isinstance(obj, list):
        return [deep_replace_values(x, replace_map) for x in obj]
    if isinstance(obj, str):
        return replace_map.get(obj, obj)
    return obj

def action_publish(ctrl: Dict[str, str]) -> Tuple[int, str, str]:

    report_glob = ctrl["s3_report_glob"]
    dest_latest = ctrl["s4_input_latest"]

    if not report_glob or not dest_latest:
        return (EC_CONFIG, "Publish 실패: s3_report_glob 또는 s4_input_latest가 비었습니다.", "")

    pattern = to_abs(root, report_glob)
    candidates = list(pattern.parent.glob(pattern.name))
    if not candidates:
        return (EC_INPUT_MISSING, f"Stage3 리포트 없음: {pattern}", "")

    latest = max(candidates, key=lambda p: p.stat().st_mtime)

    dest = to_abs(root, dest_latest)
    ensure_dir(dest.parent)

    # backup existing dest
    if dest.exists():
        backup_dir = dest.parent / "_backup"
        backup_file(dest, backup_dir)

    shutil.copy2(latest, dest)
    return (EC_OK, f"Publish 완료: {latest.name} → {dest}", str(dest))


def run_subprocess(script_abs: Path, cwd: Path, log_file: Path, args_list: list[list[str]]) -> Tuple[bool, str]:
    """
    Try multiple command variants until one succeeds.
    Writes stdout/stderr to log file.
    """
    for cmd in args_list:
        try:
            p = subprocess.run(
                cmd,
                cwd=str(cwd),
                capture_output=True,
                text=True,
                check=False,
            )
            log_file.parent.mkdir(parents=True, exist_ok=True)
            with log_file.open("a", encoding="utf-8") as f:
                f.write("\n" + "=" * 90 + "\n")
                f.write("CMD: " + " ".join(cmd) + "\n")
                f.write(f"RC={p.returncode}\n")
                f.write("--- STDOUT ---\n" + (p.stdout or "") + "\n")
                f.write("--- STDERR ---\n" + (p.stderr or "") + "\n")

            if p.returncode == 0:
                return (True, "OK")
            # if looks like argument mismatch, try next variant
            bad_arg = ("unrecognized arguments" in (p.stderr or "").lower()) or ("usage:" in (p.stderr or "").lower())
            if bad_arg:
                continue
            # otherwise stop early
            return (False, f"RC={p.returncode}")
        except Exception as e:
            with log_file.open("a", encoding="utf-8") as f:
                f.write("\n" + "=" * 90 + "\n")
                f.write(f"EXCEPTION: {e}\n")
            return (False, str(e))

    return (False, "All command variants failed.")

def main(argv: list[str] | None = None) -> int:

        return EC_INPUT_MISSING

1. **엑셀 파일을 `.xlsm`로 저장**

* 추천: 프로젝트 폴더 안(= `ProjectRoot`)에 `HVDC_PIPELINE_CONTROL.xlsm` 형태로 저장

2. VBA 모듈 추가

* `ALT+F11` → VBA 편집기
* 표준 모듈 3개 생성 후 위 코드 붙여넣기

  * `mod_CONTROL_Setup`, `mod_CONTROL_Run`, `mod_UTILS`

3. CONTROL 템플릿 생성

* 매크로 실행: `BTN_SetupControl`
* CONTROL/LOG 시트가 자동 생성되고 버튼도 배치됩니다.

4. Python runner 배치

* 프로젝트 루트 기준 `scripts\run_pipeline.py`로 저장(위 Python 코드)
* CONTROL!B6(`n_RunPipelinePy`)가 정확히 그 경로를 가리키도록 확인

5. Python 패키지 설치

* (가상환경 권장)

  * `pip install openpyxl ruamel.yaml`
  * Stage2가 import로 실행되므로 프로젝트가 `scripts/...` 형태로 import 가능해야 합니다(위 runner가 `sys.path`에 root를 추가).

6. 운영 순서(버튼만 클릭)

* **💾 YAML 반영(백업)** → Stage1 → Stage2 → Stage3 → **📌 Publish** → Stage4
* 또는 **▶ RUN ALL (1→4)**

## 5) 검증·주의·옵션

### A) “YAML 편집 없이 Excel만으로 운영” 검증 포인트

* `💾 YAML 반영(백업)` 클릭 시

  * 각 YAML 옆에 `_backup` 폴더가 생성되고 timestamp 백업 파일이 남아야 합니다.
* Stage2 YAML은 문서 구조(`input.synced_file`, `output.derived_file`)가 고정이라 반영이 확실합니다.
* Stage3 YAML은 `stages.stage3.io.derived_file`, `report_file`이 고정입니다.

### B) Publish 버튼(최신 Stage3 리포트 자동 탐색→Stage4 input)

* Stage3 출력은 glob(`HVDC_입고로직_종합리포트_*.xlsx`) 형태이며 12개 시트가 생성됩니다.
* Stage4는 Stage3 리포트의 `통합_원본데이터_Fixed`를 사용합니다.
* 따라서 publish는 “Stage3에서 가장 최신 파일”을 찾아 Stage4 `input_file`로 복사하는 구조가 맞습니다.

### C) Stage1/3/4 실행 인자 관련(중요)

* 문서에는 핵심 스크립트 경로는 있으나(예: Stage3 `scripts/stage3_report/report_generator.py`, Stage4 `scripts/stage4_anomaly/anomaly_detector_balanced.py`) CLI 인자 규격이 명시돼 있지 않습니다.
* 그래서 `run_pipeline.py`는 **인자 2~3가지 변형을 시도하고 실패 시 무인자 실행으로 fallback**하도록 설계했습니다.

  * 만약 사내 레포에서 실제 CLI가 정해져 있다면, `variants = [...]` 부분만 1줄 수정하면 됩니다.

### D) 운영 모드 3안

* **A(권장): 로컬 운영(지금 설계)** — Excel 버튼+로컬 python
* **B: 공유폴더 운영** — ProjectRoot를 공유 드라이브로 두고, 여러 사용자가 동일 템플릿 사용
* **C: 배치 운영** — Windows Task Scheduler에서 `python run_pipeline.py run --stage stage3 ...` 형태로 야간 자동화(Excel은 모니터링 용)
