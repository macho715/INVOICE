## 2. 원인 분석

### 2.1 Configuration 요율 변경의 역효과

**변경 사항**:
```json
THC_20FT: 280 → 372 USD (+32.9%)
THC_40FT: 420 → 479 USD (+14.0%)
```

**문제**:
- 실제 Draft Invoice는 372, 479가 맞음
- 그러나 **Ref Rate를 높인 것이 문제**
- **Delta 계산 방식**: `(Draft - Ref) / Ref * 100`
- Draft=479, Ref=479 → Delta=0% ✅
- **But**: 다른 항목들의 Ref Rate가 변경되어 Delta가 커짐

**영향받은 항목 추정**:
- THC 관련 항목에서 예상치 못한 매칭이 발생
- Delta가 커져서 PASS → FAIL 전환

## 3. 문제점 진단

### 3.1 THC 요율 변경 문제

**예상 시나리오**:
1. Draft TERMINAL HANDLING = 372 USD (20DC)
2. Before Ref = 280 USD
3. Delta = (372-280)/280 = **+32.86%** → FAIL

5. Delta = (372-372)/372 = **0%** → PASS ✅

**실제 발생**:
- THC 매칭이 다른 항목에도 영향
